          # 适配等问题

          在这里推荐您使用下载本插件

          云崽问题

          可使用原版Yunzai-Botopen in new window 轻量版Yunzai-Bot-liteopen in new window 喵版Yunzai-Botopen in new window 喵崽Miao-Yunzaiopen in new window trss崽TRSS-Yunzaiopen in new window 来下载本插件

          时雨open in new window的TRSS-Yunzaiopen in new window很明显已经适配了

          协议端问题

          关于协议端推荐使用icqqopen in new window go-cqhttpopen in new window oicqopen in new window

          关于 Miao-Yunzaiopen in new window 的微信适配器：WeChat-pluginopen in new window 也将会被大大的限制使用

          关于 TRSS-Yunzaiopen in new window 的协议端：go-cqhttpopen in new window 以及 icqqopen in new window 可无限制使用该插件，其他协议端将会被大大的限制使用

          关于以上说的原因是脆脆鲨插件主要是对QQBot的管理，虽已适配但大部分功能还是限制于QQ
