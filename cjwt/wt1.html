                  # 插件报错常见解决办法

                  • 1.从plugins中删除Jinmaocuicuisha-plugin重新下载脆脆鲨插件
                  • 2.请查看你的崽是不是v3 脆脆鲨插件只支持v3
                  • 3.检查你的崽是否为原版云崽或者喵版云崽以及trss崽的go-cqhttp协议端，其他协议端未适配
                  • 4.如果是更新插件后报错了 请耐心等待会修复，如果无法更新可以再打开Jinmaocuicuisha-plugin文件运行git pull手动更新

                  # 安装插件报错

                  # 1.fatal: destination path './plugins/Jinmaocuicuisha-plugin' already exists and is not an empty directory.

                  • 这是因为在你的plugins文件中以存在Jinmaocuicuisha-plugin了

                  # 2.Cloning into './plugins/Jinmaocuicuisha-plugin'... error: RPC failed; curl 56 OpenSSL SSL_read: Connection was reset, errno 10054 fatal: expected flush after ref listing

                  • 正常因为这是github源
                  • 可以更换至gitee源下载

                  # 插件冲突问题

                  • 1.脆脆鲨插件可能会与带有打人/表情包/壁纸图片的插件产生冲突，如有发生冲突问题，删除任意一个插件后重启机器人即可
                  • 2.本插件与互殴插件（fisticuffs-plugin）会造成严重冲突 互殴插件是脆脆鲨插件的二开 有问题请找互殴插件作者本人 与脆脆鲨插件无关

                  # 脆脆鲨插件注意事项

                  • 1.使用本插件后请第一时间发送#设置绝对权限
                  • 2.请设置bot名字，指令：#设置打人Bot名字+名字
                  • 3.请写入打人api，指令：#写入打人api+api （没有api? 对着机器人发送#查看api）
                  • 4.打开锅巴找到脆脆鲨插件配置表情包token
                  • 5.选项1-3为必设置 5可以不配置不影响使用
                  • 6.一些配置可以通过锅巴配置
