    Jinmaocuicuisha-plugin(脆脆鲨插件)是Yunzai-Bot的一个功能拓展插件。

    功能示例 💡部署文档 →插件问题

    快速安装

    • 使用gitee下载：

    • git clone --depth 1 https://gitee.com/SHIKEAIXYY/Jinmaocuicuisha-plugin.git ./plugins/Jinmaocuicuisha-plugin

    • 使用github下载：

    • git clone --depth 1 https://github.com/SHIKEAIXY/Jinmaocuicuisha-plugin.git ./plugins/Jinmaocuicuisha-plugin ./plugins/Jinmaocuicuisha-plugin
